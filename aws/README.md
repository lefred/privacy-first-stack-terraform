# MariaDB Privacy-First Stack: deployment on AWS

This is the directory to use to deploy the Privacy-First Stack in Amazon Cloud:

```
cp terraform.tfvars.example terraform.tfvars
-- edit terraform.tvars with your info
terraform init
terraform plan
terraform apply
```
