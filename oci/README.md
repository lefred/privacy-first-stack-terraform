# MariaDB Privacy-First Stack: deployment on OCI

This is the directory to use to deploy the Privacy-First Stack in Oracle Cloud:

```
cp terraform.tfvars.example terraform.tfvars
-- edit terraform.tvars with your info
terraform init
terraform plan
terraform apply
```
